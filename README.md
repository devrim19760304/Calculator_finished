# Easy Calculator

A simple web-based calculator built using HTML, CSS, and JavaScript. This project demonstrates basic operations (+, -, *, /) with a user-friendly interface.

---

## Features
- Perform basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Responsive and easy-to-use interface with Bootstrap styling.
- Input validation to prevent invalid calculations.
- Clear result functionality for easy reuse.

---

## Getting Started

### Prerequisites
To run this project, you only need a web browser.

---

### How to Use
1. Open the `index.html` file in your browser.
2. Enter two numbers in the provided fields.
3. Select an operator (`+`, `-`, `*`, `/`) from the dropdown menu.
4. Click the **Calculate** button to see the result.
5. Click the **Clear Result** button to reset the result field.

---

## Files
- `index.html` : The main HTML structure of the calculator.
- `style.css` : Custom styles for the calculator layout.
- **Bootstrap CSS**: Enhances the design with responsive styling.

---

## Notes
- Division by zero will display an error message.
- The result field is read-only to prevent manual editing.
- Ensure valid numbers are entered before performing calculations.

---

## Technologies Used
- **HTML5**: For structure and layout.
- **CSS**: For styling the calculator.
- **JavaScript**: For functionality and interactivity.
- **Bootstrap**: For a modern, responsive design.
